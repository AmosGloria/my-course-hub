## course-hub assignment: Definition of Terms

## NPM(Node Package Manager): NPM is the default package manager for Node.js. NPM helps developers to install, share, and manage dependencies for their projects. It is installed in a program when Node.js is installed.

## Package.json: Package.json is a special file that describes Node.js projects. It conatins information about the app such as the name, version, dependencies, scripts and more. Package.json can be installed by running npm init in the project folder which will ask a series of questions and generate package.json. You can also run npm init -y for quick setups.

## Vite: Vite is a build tool that aims to provide a faster and leaner development experience for modern web projects.
